Open Geospatial Consortium Web Services (OWS)¶

The Open Geospatial Consortium (OGC) is a standards organization that has established web service (OWS) protocols for the exchange and processing of spatial data.

The OWS address for Mitwelten is:

https://geo.mitwelten.org/geoserver/ows

Table Joining Service (TJS)¶

The Table Joining Service (TJS) is a standard for the exchange of tabular geospatial data.

Web Coverage Service (WCS)¶

The Web Coverage Service (WCS) is a standard for the exchange of raster geospatial data.

Web Feature Service (WFS)¶

The Web Feature Service (WFS) is a standard for the exchange of vector geospatial data.

Web Processing Service (WPS)¶

The Web Processing Service (WPS) is a standard for the exchange of inputs and outputs for geospatial processing.
