Conneting to the Web Coverage Service (WCS)¶

  1. Select Layer > Data Source Manager from the menu or press Ctlr+L

  2. Select WCS from the left panel

  3. Click New in right panel or press Alt+N to define a new connection

  4. Enter a meaningful name such as Mitwelten OWS and the URL https://fhnw-imvs-afe077e3-1c48-407a-b75c-c8cb4a9076a0.try.yaler.io/geoserver/ows

  5. Click OK or press Alt-O to save the connection details

  6. Select the name for the connection from the dropdown menu in the right panel and click Connect or press Alt+O

The list of available layers will appear.

Conneting to the Web Feature Service (WFS)¶

  1. Select Layer > Data Source Manager from the menu or press Ctlr+L

  2. Select WFS/OGC API - Features from the left panel

  3. Click New in right panel or press Alt+N to define a new connection

  4. Click New in right panel or press Alt+N to define a new connection

  5. Enter a meaningful name such as Mitwelten OWS and the URL https://fhnw-imvs-afe077e3-1c48-407a-b75c-c8cb4a9076a0.try.yaler.io/geoserver/ows

  6. Click OK or press Alt-O to save the connection details

  7. Select the name for the connection from the dropdown menu in the right panel and click Connect or press Alt+O

The list of available layers will appear.
