OGR2PG2GS¶

Store vector data from a file in a PostGIS database and publish it to GeoServer

usage: ogr2pg2gs [-h] [-v] [-l] [-dt] name path

Positional Arguments¶

name

The table name to use for the vector data in PostGIS

path

Vector data to store in PostGIS and publish to GeoServer

Named Arguments¶

-v, --version

show program version

Default: False

-l, --log

Log activity to the specified file

Default: False

-dt, --droptable

Drop table if it already exists

Default: False
